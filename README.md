# 🧺 Laundry Booking System (MERN Stack)

## 📌 Overview

A full-stack web application for managing laundry service bookings with dynamic pricing and scheduling. Users can create bookings, select services, and track order status.

---

## 🚀 Features

* Create and manage laundry bookings
* Dynamic pricing based on service type and items
* RESTful API for booking operations
* Booking status tracking (pending, in-progress, completed, cancelled)
* Input validation and error handling

---

## 🛠️ Tech Stack

* Frontend: React.js
* Backend: Node.js, Express.js
* Database: MongoDB, Mongoose
* Validation: Joi

---

## ⚙️ How It Works

### Backend:

* Handles booking logic and API endpoints
* Stores data in MongoDB
* Calculates price based on selected services

### Frontend:

* User interface for booking and viewing services
* Sends requests to backend APIs

---

## ▶️ How to Run

### 1. Backend

```bash
cd backend
npm install
npm start
```

### 2. Frontend

```bash
cd frontend
npm install
npm start
```

---

## 🧠 Architecture

Frontend (React) → Backend (Express API) → MongoDB Database
